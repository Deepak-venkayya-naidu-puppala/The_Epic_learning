Datatypes are used to specify the type of 
data.
datatypes in c are mainly classified into
2 types.
    1.primary datatypes.
    2.secondary datatypes.
primary datatypes:

The datatypes which are already defined 
in the language.
These are about to known as basic/
elementary/primitive/preliminary datatypes
the various available primary datatypes
available in c are:
 

  char
  int
  float
  double
  void
secondary datatypes:

the datatypes which can be defined by the 
user and formed with the combination of 
primary datatypes
   eg::
    Arrays
    structures
    unions
    pointers etc.,
